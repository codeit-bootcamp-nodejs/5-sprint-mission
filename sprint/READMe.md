### 목표 API 리퀘스트 보내는 코드 구현

### Entity
+ Article, Product, ElectronicProduct class를 각 파일로 생성했습니다.

### Service
+ ArticleService.js 파일에는 Aritcle API 요청 함수를 구현했습니다.
+ ProductService.js 파일에는 Product API 요청 함수를 구현하고, getProductList 함수를 통해 받아온 상품 리스트를 Product와 ElectronicProduct 클래스를 사용해 인스턴스로 생성했습니다.

### main.js
+ main.js 파일에는 함수 호출부를 작성했으며, 동작 여부를 확인했습니다.
